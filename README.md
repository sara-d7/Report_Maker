This program makes the necessary changes to the structural analysis and design report for the submission/approval to the municipality of Nepal.
The program takes over the repetitive mundane works while making the report.
Automatically places the snapshots of the etabs analysis screenshots at the requried positions in the report provided the screenshots are already saved in a separate folder with the correct filename.
Links the design excel sheets at the required section of report.
Saves time in the report making process.
Final report should be skimmed through manually after the program is run successfully.
